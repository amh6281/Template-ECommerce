  ## 📝 About The Project
> <b>해당 프로젝트는 일반 사용자의 상품 구매, 쇼핑몰 관리자의 상품 관리 등 일반적인 구조의 쇼핑몰이 아닌, 더 나아가 사업자 회원이 쇼핑몰 개설 버튼을 통해 필요한 정보를 기입하고, 원하는 쇼핑몰 디자인의 템플릿과 판매 카테고리를 선택하여 웹사이트 내에 무료로 쇼핑몰을 개설할 수 있는 웹사이트입니다.</b>
  <br />
